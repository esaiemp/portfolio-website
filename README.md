# Esaie Portfolio Template

A responsive, single-page portfolio website designed for both job applications and consulting opportunities.

## Run locally

Open `index.html` directly in your browser, or use VS Code Live Server.

## Main files

- `index.html` — content and page structure
- `styles.css` — visual design and responsive layout
- `script.js` — mobile menu, animations, and demo contact-form behaviour

## Recommended customizations

1. Replace the LinkedIn and GitHub links in the footer.
2. Update the project descriptions with verified metrics and outcomes.
3. Add a real profile photo or professional portrait.
4. Connect the contact form to Formspree, EmailJS, n8n, or your own API.
5. Add a downloadable PDF résumé.
6. Replace placeholder industry and experience details where necessary.

## Deploy

This template can be deployed as-is to:

- GitHub Pages
- Netlify
- Vercel
- Azure Static Web Apps

## Contact form integration

The current form is a front-end demo. For an easy production setup, connect it to Formspree or trigger an n8n webhook.
